# Стек и очередь: объяснение

## Стек (Stack)

**Стек** — это структура данных, работающая по принципу LIFO (Last In, First Out — «последним пришёл, первым ушёл»).

### Операции со стеком:
- **Push** — добавить элемент на вершину стека
- **Pop** — удалить элемент с вершины стека
- **Top/Peek** — посмотреть элемент на вершине без удаления

### Иллюстрация стека:

```
    ┌─────┐
    │  5  │  ← TOP (Push 5)
    ├─────┤
    │  3  │
    ├─────┤
    │  1  │  ← BOTTOM
    └─────┘

После Pop:  элемент 5 удалён, 3 стал TOP
```

### Аналоги из жизни:
- **Стопка тарелок**: тарелки кладутся друг на друга, взять можно только верхнюю
- **Колодка карт**: добавляем карты сверху, достаём тоже сверху
- **Браузер "Назад"**: история посещений — последний открытый сайт первым показывается

## Очередь (Queue)

**Очередь** — это структура данных, работающая по принципу FIFO (First In, First Out — «первым пришёл, первым ушёл»).

### Операции с очередью:
- **Enqueue** — добавить элемент в конец очереди
- **Dequeue** — удалить элемент из начала очереди
- **Front** — посмотреть первый элемент без удаления

### Иллюстрация очереди:

```
До Enqueue(4):        После Enqueue(4):
┌───┬───┬───┬───┐    ┌───┬───┬───┬───┬───┐
│ 1 │ 2 │ 3 │   │    │ 1 │ 2 │ 3 │ 4 │   │
└───┴───┴───┴───┘    └───┴───┴───┴───┴───┘
  ↑                  ↑
FRONT               FRONT

После Dequeue:       (1 удалён)
┌───┬───┬───┬───┬───┐
│   │ 2 │ 3 │ 4 │   │
└───┴───┴───┴───┴───┘
  ↑
FRONT
```

### Аналоги из жизни:
- **Очередь в магазине**: первым пришёл — первым обслужили
- **Очередь печати**: документы печатаются в порядке отправки
- **Банковская очередь**: кто раньше пришёл, того раньше обслужат

## Сравнение

| Характеристика | Стек | Очередь |
|---------------|------|---------|
| Принцип | LIFO | FIFO |
| Вход/выход | Один конец | Два конца |
| Пример жизни | Стопка тарелок | Очередь в кассу |

---

## Реализация в программировании

### Стек ( массив ):
```c
int stack[100];
int top = -1;
stack[++top] = value;  // Push
value = stack[top--];  // Pop
```

### Очередь ( массив ):
```c
int queue[100];
int front = 0, rear = -1;
queue[++rear] = value;        // Enqueue
value = queue[front++];       // Dequeue
```
